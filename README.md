## 📖 나랏ᄆᆞᆯᄊᆞᆷ이 ᄑᆞ이썬(Python - *Ye Olde Lang*)ᅢ 다라
*(Narammai-ssami Python-ae dara)*

리호세 마스터(Maister Hose)ᅵ 이를 위ᄒᆞ야 어엿비 너겨 ᄉᆡ로 셈글 비기(Semgle Secret Writ)를 ᄆᆡᇰᄀᆞ노니, (Maister Hose hath made this for thy mercy)
